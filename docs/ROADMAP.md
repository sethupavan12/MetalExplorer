# Roadmap

This is the public direction, not a promise.

## 0.1

- Local process dashboard.
- Services, Agents, Cleanup, and Network views.
- Guarded termination.
- OpenAI-compatible AI explanations.
- Light, dark, and Matrix themes.
- Safety and privacy documentation.

## 0.2

- Signed and notarized public macOS releases.
- Intel or universal artifact verification.
- Homebrew cask.
- Advanced network details toggle.
- Better process grouping for Electron, browser, and agent helper trees.
- Safer command redaction before AI explanations.

## 0.3

- Process tree view.
- Per-project local services grouping.
- More explainability rules for MCP, package runners, and local model servers.
- Exportable diagnostics bundle with redaction.
- Accessibility pass.

## Later

- Optional local-only model explanation mode.
- Menu bar mini monitor.
- LaunchAgent detection.
- Login item detection.
- Signed update feed.

## Non-goals

- Malware verdicts without evidence.
- Silent telemetry.
- Automatic AI calls.
- Packet capture.
- Root process termination.
- Background daemon by default.
